# AI-Netflix-Recommendation-Agent
An Learning-based AI agent that monitors users preferences and recommend the movies and series based on that along with that best frontend.

It focuses on watch time, number of views, like & dislikes and search feature...

For making it to run, install python libraries used in main file and then run:
1. python -m http.server 8000 (In first Terminal)
2. python app.py (In second terminal)
3. Type in browser: localhost:8000
